# Under Vibes Web

Sitio web dedicado a la cultura hip hop latinoamericana. Proyecto académico con estructura avanzada en HTML, CSS y Bootstrap.

## 🔧 Tecnologías usadas
- HTML5 semántico
- CSS3 personalizado
- Bootstrap 5.3.2
- Git y GitHub para control de versiones
- GitHub Pages para publicación
